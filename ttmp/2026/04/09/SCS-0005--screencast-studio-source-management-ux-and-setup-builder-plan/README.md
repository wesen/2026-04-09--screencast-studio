## Getting Started

Use docmgr commands to manage this workspace:

- Add documents: `docmgr doc add --ticket SCS-0005 --doc-type design-doc --title "My Design"`
- Import sources: `docmgr import file --ticket SCS-0005 --file /path/to/doc.md`
- Update metadata: `docmgr meta update --ticket SCS-0005 --field Status --value review`
