# Screencast Studio UI

Component library for the Screencast Studio web control frontend. Based on the
visual design from `screencast-studio-v2.jsx.jsx`.

## Design Tokens

```css
--studio-cream: #f5f0e8;
--studio-black: #1a1a1a;
--studio-dark: #2c2c2c;
--studio-mid: #8a8a7a;
--studio-light: #d4d0c8;
--studio-red: #c04040;
--studio-green: #5a8a5a;
--studio-amber: #b89840;
--studio-bg: #e8e4dc;
```

Font: `"Chicago", "Geneva", "Monaco", monospace`

## Stories

### Primitives
- **Button** - `Studio/Button` - Button with active/accent states
- **Radio** - `Studio/Radio` - Radio indicator
- **Sel** - `Studio/Sel` - Custom dropdown selector
- **Slider** - `Studio/Slider` - Range slider
- **WinBar** - `Studio/WinBar` - Window title bar with stripe pattern
- **Win** - `Studio/Win` - Window container

### Source Components
- **FakeScreen** - `Studio/FakeScreen` - Source preview visualization
  - Display, Window, Region, Camera variants
- **MicMeter** - `Studio/MicMeter` - Audio level meter
- **Waveform** - `Studio/Waveform` - Audio waveform display
- **SourceCard** - `Studio/SourceCard` - Individual source with controls
- **AddSourceButton** - `Studio/AddSourceButton` - Add new source
- **SourceGrid** - `Studio/SourceGrid` - Grid of source cards

### Studio Panels
- **OutputPanel** - `Studio/OutputPanel` - Output parameters and transport
- **MicPanel** - `Studio/MicPanel` - Microphone controls
- **StatusPanel** - `Studio/StatusPanel` - Recording status
- **MenuBar** - `Studio/MenuBar` - Application menu
- **StudioPage** - `Pages/StudioPage` - Main studio screen layout

## Development

```bash
# Install dependencies
cd ui && pnpm install

# Start development server
pnpm dev

# Start Storybook
pnpm storybook

# Build for production
pnpm build
```

## Architecture

- **RTK Query** for API data fetching
- **Redux Toolkit** for local state
- **MSW** for API mocking
- **CSS Modules** via BEM-lite class naming
